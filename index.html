Информация для разработчиков

Инструменты

По умолчанию на проекте установлены PUG для формирования html, SASS - для формирования стилей, GULP - для сборки проекта.

assets

содержит статичные некомпилируемые папки img (изображения), vendor (сторонние библиотеки), fonts (файлы подключенных шрифтов)

build

Содержимое папки build является результатом сборки файлов из папки _source.

_source

Папка _source содержит исходные файлы разметки, скриптов и стилей. blocks - отдельные файлы разметки, скриптов и стилей под каждый блок; svg - исходные файлы для svg-спрайта; js, pages, sass - общие файлы проекта.

Getting started

gulp - запуск сборки с отслеживанием изменений;

gulp --local - запуск сборки с отслеживанием изменений и трансляцией в браузер;

gulp --build-only - единоразовый запуск сборки без отслеживания изменений.
